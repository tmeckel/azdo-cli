## azdo
Work seamlessly with Azure DevOps from the command line.
### Core commands
* [azdo auth](./azdo_auth)
* [azdo project](./azdo_project)
* [azdo repo](./azdo_repo)

### Additional commands
* [azdo config](./azdo_config)

### Options


* `--version`

	Show azdo version


### Examples

```bash
$ azdo project list
$ azdo repo clone myorg/myrepo
```
