## Command `azdo boards area`

Manage area paths used by Azure Boards.

### Available commands

* [azdo boards area project](./azdo_boards_area_project.md)

### ALIASES

- `a`

### Examples

```bash
# List the area paths for a project
azdo boards area project list Fabrikam
```
